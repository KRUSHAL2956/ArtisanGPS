============================================================
MARKETINTEL AI – WEBSITE GENERATION RULE SYSTEM
FINAL DESIGN DISCIPLINE (WHITE MINIMAL PREMIUM THEME)
============================================================

CORE THEME DIRECTION
------------------------------------------------------------
Theme Type:
White Minimal • Premium SaaS • Professional Intelligence Platform

Design Inspiration:
Clean white SaaS style (like Prosperico reference)
Soft shadows, large rounded dashboard preview, balanced spacing.

Tone:
Premium • Analytical • Corporate-grade • Portfolio-ready


------------------------------------------------------------
1. COLOR SYSTEM (STRICT)
------------------------------------------------------------

Root Brand Color:
#0B3C5D (Deep Data Blue)

Accent:
#00A8A8 (Teal – subtle highlights only)

Secondary Text:
#2F4F4F (Slate Gray)

Background:
#F5F7FA (Light Neutral)

Card Background:
#FFFFFF

Border:
#E3E8EF

Warning:
#FFB703

Danger:
#E63946

Positive:
#2E7D32

Rules:
- Maximum 3 brand colors per page
- No dramatic gradients
- No purple/pink AI gradients
- Only subtle section background shift (very light gray tone)
- No glow effects


------------------------------------------------------------
2. FORBIDDEN DESIGN ELEMENTS
------------------------------------------------------------

- No emojis
- No AI-style illustrations
- No cartoon vector humans
- No neon glow
- No glassmorphism
- No floating abstract shapes
- No overused startup gradient blobs
- No template hero background patterns
- No rainbow charts
- No excessive shadows


------------------------------------------------------------
3. LAYOUT SYSTEM
------------------------------------------------------------

Max Width:
1200–1280px

Spacing System:
8px base grid

Card Radius:
12–16px

Card Padding:
24px standard

Shadow:
Soft, subtle only

Structure Per Page:
Signal Zone
Insight Zone
Action Zone

Every page must have:
ONE dominant visual element (usually a chart or dashboard preview).

Avoid:
Equal visual weight for all components.


------------------------------------------------------------
4. NAVIGATION DESIGN
------------------------------------------------------------

Navbar:
- White background
- Subtle bottom border
- Logo left
- Menu right
- Primary button solid deep blue
- No transparency blur

Sidebar (App):
- Minimal
- Icon + label
- Active highlight with left border
- No heavy animation


------------------------------------------------------------
5. TYPOGRAPHY RULES
------------------------------------------------------------

Font:
Inter (single font family only)

Hierarchy:
H1: 48–56px bold
H2: 32–36px semi-bold
H3: 20–24px medium
Body: 16px regular
Caption: 14px muted

Rules:
- No decorative fonts
- No mixed font families
- Strong hierarchy
- Proper line height
- Clean paragraph spacing


------------------------------------------------------------
6. COMPONENT DESIGN RULES
------------------------------------------------------------

Cards:
- White
- Light border
- Balanced spacing
- No stacking shadows

Buttons:
- Primary: solid #0B3C5D
- Secondary: outline
- Hover: slight darken only
- No gradient buttons
- No glowing hover

Icons:
- Lucide or Heroicons only
- Line-based style
- Consistent stroke width
- Professional minimal look


------------------------------------------------------------
7. CHART DESIGN RULES (Chart.js)
------------------------------------------------------------

Primary Line:
Deep Blue (#0B3C5D)

Forecast Line:
Teal dashed

Confidence Band:
Low opacity teal

Grid Lines:
Light gray only

No:
- Bright rainbow colors
- Thick heavy lines
- Over-decorated tooltips

Charts must feel:
Financial-grade, analytical, clean.


------------------------------------------------------------
8. ANIMATION RULES (PREMIUM SUBTLE ONLY)
------------------------------------------------------------

Allowed:
- Fade-in on scroll (subtle)
- 200–300ms ease transitions
- Card hover slight elevation
- Button hover smooth color shift
- Chart load animation

Not Allowed:
- Bounce
- Spin
- Floating elements
- Parallax overload
- Dramatic movement

Animation must feel:
Calm • Premium • Intentional


------------------------------------------------------------
9. LANDING PAGE STRUCTURE
------------------------------------------------------------

Navbar

Hero Section:
- Left: Headline + Subtext + Buttons
- Right: Large rounded dashboard preview
- Clean whitespace
- No background illustration

Core Intelligence Section:
- 3 structured feature cards
- Equal height
- Minimal icons

Decision Intelligence Section:
- 2 large cards side-by-side
- Production + Cost Guard

Architecture Section:
- Clean line-based diagram
- No decorative illustration

Final CTA Section:
- Centered
- Simple
- Strong but minimal


------------------------------------------------------------
10. UNIQUENESS RULE
------------------------------------------------------------

To avoid AI-generated look:

- Custom spacing decisions
- Asymmetrical but balanced layout
- Avoid generic 3-column hero clichés
- Avoid repetitive identical cards
- Use intentional grouping

Website must look:
Human-designed
Engineered
Intentional
Production-ready


------------------------------------------------------------
11. BRAND POSITIONING
------------------------------------------------------------

MarketIntel AI is:

AI-Driven Market Intelligence & Decision Support Platform

It must visually communicate:

Data → Prediction → Decision → Explanation

It must look suitable for:

AI Engineer Portfolio
ML Internship Applications
Startup Demo
Enterprise Client Presentation
